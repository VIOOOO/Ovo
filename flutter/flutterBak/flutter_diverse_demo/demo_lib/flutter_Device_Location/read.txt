Flutter Native Device 中获取设备信息 Location 地理位置 
	
	从 Flutter 应用程序中获取当前设备信息
		pubspec.yaml ==> dependencies:
							device_info: ^0.4.1+4
							
	地理定位 Flutter  提供的官方定位是国外位置 location，国内无法使用，
	所以使用高德地图代替，
		要使用高德地图定位,首先需要申请 Key:
		1、申请成为开发者
		
		2、创建应用配置获取 Key （参考教程演示）
			https://lbs.amap.com/api/android-sdk/guide/create-project/get-key
			信息填写需要应用包名 packageName，应用包名所在位置：
			==> D:\Android\Project\flutter_app_test\android\app\src\main\AndroidManifest.xml
			==> <manifest xmlns:android="http://schemas.android.com/apk/res/android" package="com.example.flutter_app_test">
			应用包名字为：com.example.flutter_app_test
			
		3、申请到秘钥之后 安装依赖
		pubspec.yaml ==> dependencies:
							amap_location: ^0.2.0
							
		4、修改你的项目录 /app/build.gradle 在 android/defaultConfig 节点修改 
			manifestPlaceholders, 新增高德地图 key 配置
			
			android {
				.... 你的代码
				defaultConfig {
					.....
					manifestPlaceholders = [
						AMAP_KEY : "aa9f0cf8574400f2af0078392c556e25", // 高德地图 key
					]
				}
				...你的代码
				dependencies {
					/// 注意这里需要在主项目增加一条依赖，否则可能发生编译不通过的
					情况
					implementation 'com.amap.api:location:latest.integration'
					...你的代码
				}
				
		5、android 6 以上需要获取定位权限，需要再加入插件文档内的代码，在注意点栏的代码
